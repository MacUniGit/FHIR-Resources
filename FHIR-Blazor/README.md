## [C\#](https://github.com/s23m/roma/tree/main/FHIR-Blazor) Local Deployment

Clone the project

```bash
  git clone https://github.com/s23m/roma
```

Go to the project directory

```bash
  cd roma/FHIR-Blazor
```

Start the application

```bash
  dotnet run
```
